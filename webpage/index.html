RAG Assistant

A production-ready Retrieval-Augmented Generation system that answers questions grounded in your own documents — not the model's parametric memory.

  • Ingests text documents, chunks with 200-token overlap, and indexes OpenAI embeddings in a persistent JSON vector store
  • Vectorised cosine similarity search (pre-normalised matrix multiply) retrieves the top-k chunks at query time
  • Per-user conversation memory (last 10 exchanges) fed into the GPT-3.5-turbo context window alongside retrieved chunks
  • FastAPI backend + Streamlit UI with source attribution and confidence scoring on every response

Computer Vision — Object Detection

Real-time object detection service built on Faster R-CNN ResNet-50 FPN, served through a FastAPI backend with a drag-and-drop web interface.

  • Loads a checkpoint or falls back to COCO pretrained weights (80+ classes); GPU/CPU auto-detected at startup
  • Preprocessing transform built once at init — not on every request — keeping per-image overhead minimal
  • Confidence threshold and NMS IoU threshold configurable per request via query parameter
  • Structured DetectionResult dataclass with bounding boxes, class labels, scores, and inference timing

AI Resume Screening

NLP pipeline that ranks candidates against job descriptions using transformer embeddings, with rigorous data quality checks at every stage of ingestion.

  • Parses PDF and plain-text resumes; generates sentence-transformer embeddings and ranks candidates by cosine similarity to the job description
  • Pydantic v2 schemas enforce strict validation of contact info, experience dates, skills count, and education across every record
  • DataQualityChecker reports completeness %, duplicate rate, and consistency issues before any model sees the data
  • REST API with structured CandidateMatch responses including rank, score, tier, and recommendation text

RentPulse

A live Chrome extension that monitors multiple Irish rental listing sites and sends instant desktop alerts for new properties — published to the Chrome Web Store with real users.

  • Polls Daft.ie, Rent.ie, and MyHome in the background; triggers desktop notifications the moment a new listing matches saved filters
  • Supabase PostgreSQL backend stores listings, user preferences, and deduplication state
  • Scam-detection heuristics flag suspicious listings before they reach the user
  • Node.js cloud API hosted on Render; React + Vite frontend with full user account management
